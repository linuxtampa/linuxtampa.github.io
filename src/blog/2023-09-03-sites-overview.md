I have owned and operated a number of websites over the years. Here is a list of them, with a brief description of each.

* [bailey-jones.com](https://bailey-jones.com) — my personal/music site, started in 2019. It is a static site built with Eleventy.
* [paperprinciples.com](https://paperprinciples.com) — a website for my late wife's origami crafts. It is a static site, with embedded youtube videos we recorded together.
* [gotonight.com](https://gotonight.com) — a website for a startup I worked for, started in 2012. It was written in Coldfusion, and has been redeveloped in React.
* [tuckasee.com](https://tuckasee.com) — This is a site for my father's fine handcrafted bowed psalteries and hammered dulcimers.   He retired many years ago, and passed away in March 2026, but I keep this site up as a tribute to him.
* [sunshinebrass.com](https://sunshinebrass.com) — a website for a brass band, started in 2005. It was written in PHP, and is now a static site.
* [gasparbrass.com](https://gasparbrass.com) — a website for my brass quintet, started in 2005. It was written in PHP, and is now a static site.
* [adamjones.tech](https://adamjones.tech) — my son's website, started in 2019. It is a static site, done in React.
* [timjones.com](https://timjones.com) — my personal website, started in 2015. Now redirects to bailey-jones.com.
* [tampabaybrass.com](https://tampabaybrass.com) — a website for a brass quintet, started in 2000. 
* [carrollwoodbrass.com](https://carrollwoodbrass.com) — a website for a brass ensemble, started in 2012.
