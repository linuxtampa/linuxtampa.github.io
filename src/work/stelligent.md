# Stelligent / Mphasis (2017–2020)

Stelligent was the most impactful job change of my career — my first fully remote role and my entry into public cloud infrastructure. Their onboarding program alone was worth the move: a structured 90-day curriculum built around the public [Stelligent U](https://github.com/stelligent/stelligent-u) repo, with a mentor guiding each new hire through AWS fundamentals.

By the 90-day mark I had earned my **AWS Certified Developer Associate**. A year or so later, the **AWS DevOps Professional**.

> [Mphasis acquired Stelligent in November 2018](https://www.mphasis.com/content/dam/mphasis-com/global/en/news/press_releases/Press-Release-Mphasis-acquires-Stelligent-Nov-8-2018.pdf). My work continued unchanged under the Stelligent brand.

## AQR Capital (Greenwich, CT)

AQR was a quantitative investment firm running a large, security-conscious AWS environment. Key work:

- **AWS Batch** automation to package hundreds of one-off data-science Python programs into limited, secure containers
- **AWS Service Catalog** for governed, self-service infrastructure provisioning
- **Python/boto3** scripting as a credible alternative to CloudFormation for certain deployment patterns
- **AWS ACM-PCA** to programmatically provision and sign thousands of distinct SSL certificates, dramatically limiting the attack surface in the event of an exploit

## 3M (Maplewood, MN)

Always an innovation leader, as well as a national industrial icon, 3M was exploring the AWS IoT suite for collecting real-time data from manufacturing equipment - furnaces, grinders - not customer products. My work:

- Built a **React + Python/Flask** app to create and edit production "recipes" for manufacturing machines
- Helped architect real-time IoT data collection pipelines
- **Trained ~20 developers** in Python and AWS fundamentals
